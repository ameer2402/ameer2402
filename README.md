## 🚀 Featured Projects

### 🎓 [VedaMate – Multi-Agent RAG AI Teaching Assistant](https://github.com/ameer2402/VedaMate)
> 🏅 **Showcased at Google AI India Hackathon 2025 (Onsite)**

A production-grade Conversational AI Teaching Assistant that lets students upload **200+ page textbook PDFs** and engage in intelligent, context-aware conversations grounded in their study material combined with real-time web data.

**🧠 Architecture:** Designed a multi-agent **LangGraph workflow** orchestrating **6+ specialized AI agents** — Query Rewriter, Multi-Query Generator, Router Agent, Vector DB Agent, Web Search Agent, and Response Synthesizer.

**⚡ Key Innovation:** Hybrid parallel search architecture using Python `asyncio` querying ChromaDB and Google Custom Search API simultaneously, reducing latency by **40%**.

**📊 Impact Metrics:**
- ✅ Improved answer relevance by **50%**
- ✅ Reduced hallucinations by **30%**
- ✅ Optimized token usage by **25%** via intent classification
- ✅ Generated 10–20 isolated topic modules per textbook with auto-quiz generation

`Python` `LangGraph` `LangChain` `Google Gemini` `RAG` `Multi-Agent Systems` `ChromaDB` `FastAPI` `Streamlit` `Prompt Engineering`

---

**✨ Key Features:** Quill rich-text editor, category-based search, real-time commenting, automated email notifications, Cloudinary image storage, Multer file uploads, and fully responsive Bootstrap UI.

**📊 Impact:**
